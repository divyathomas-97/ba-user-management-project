# BA Project – User Management System

## Overview
This project demonstrates Business Analyst activities for a User Management System including requirement gathering, documentation, and validation.

## Documents Included
- BRD (Business Requirement Document)
- SRS (Software Requirement Specification)
- User Stories
- Acceptance Criteria
- Test Cases
- Flow Diagram

## Key Features
- Requirement Analysis
- Documentation (BRD & SRS)
- User Story Creation
- Functional Validation through Test Cases
- System Flow Design

## Tools Used
- MS Word / Google Docs
- Excel
- Draw.io

## Purpose
To showcase Business Analyst skills including requirement analysis, documentation, and communication between business and technical teams.
